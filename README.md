Title: Privacy Policy for CYBERSCAN
Effective date: March 5, 2026
CYBERSCAN is a browser extension that analyzes websites selected by the user to identify security issues (for example exposed API keys, insecure cookie settings, and related client-side risks).
1) Data We Collect
CYBERSCAN is designed to minimize data collection.
Website security scan data: URL/domain, request/response metadata needed for security analysis, cookie attribute metadata (e.g., Secure/HttpOnly/SameSite), and detected findings.
Extension settings: user preferences and local scan configuration.
Diagnostics (optional): error logs if implemented for troubleshooting.
2) How Data Is Used
We use data only to:
run security scans requested by the user,
generate and display findings/remediation suggestions,
store settings and local history for user convenience,
improve extension reliability and performance.
3) Data Sharing
We do not sell personal data.
We do not share scan data with third parties except:
when required by law, or
if explicitly disclosed in a future update and accepted by users.
4) Data Storage and Retention
By default, settings and scan-related data are stored locally in the browser extension storage.
If cloud/back-end processing is introduced in future versions, this policy will be updated before release.
5) Permissions
CYBERSCAN requests only permissions required for its single purpose (website security analysis), including activeTab, scripting, tabs, cookies, webRequest, storage, and required host permissions.
6) Security
We take reasonable measures to protect data handled by the extension and limit access to only what is necessary for scanning functionality.
7) User Controls
Users can:
remove the extension at any time,
clear extension storage via browser settings,
stop using scan features at any time.
8) Changes to This Policy
We may update this Privacy Policy as the extension evolves. Updates will be reflected by revising the effective date.
